Oregairu SNAFU

Source Code Android Download
2019 ~ 2020
Java
Android Studio

This is an Android Application inspired by the Oregairu series.

This application offers a diverse set of features, including a Music Player and a Book Reader. It also contains a Mini-Game where users can click on a character, causing them to perform a random animation accompanied by a corresponding sound. This concept is inspired by the popular application Tuturu.
