This is my first 3d Portfolio Website in which I built my own world.

On this website, I created my own little Galaxy where each planet represents a project I made. Therefore, to discover my projects, we can explore the galaxy by navigating through the planets.

However, I decided to not make it my main portfolio website. I felt like the world I built focused too much on its environment, and did not highlight my projects well enough (which is the main goal of a portfolio). Furthermore, since it's a 3d website, it did take a bit longer to load, which might discourage some visitors. I did make an overview menu to navigate through the projects faster, but I didn't want to overwhelm the users with too much information right at the start.

Nevertheless, this project allowed me to learn a lot about TypeScript, and how a 3d Library like ThreeJs works. Furthermore, ensuring the best user experience was very important to me. Thus, I did a lot of Optimization to make everything fast, and fixed a lot of sketchy bugs along the way.

Although I don't update it anymore, you can still visit my 3d Portfolio Website.
